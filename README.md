# Queue Manager (Менеджер очереди)

Проект на React для управления очередью задач с возможностью экспорта в JSON.

## Функционал
- Добавление задач с указанием приоритета, срока и исполнителя.
- Автоматическое сохранение данных в `localStorage`.
- Копирование очереди в формате JSON.
- Скачивание данных в виде `.json` файла.
- Современный адаптивный дизайн с анимациями.

## Установка и запуск
1. Установите зависимости: `npm install`
2. Запустите проект: `npm run dev`

## Технологии
- React
- Vite
- Lucide React (иконки)
- React Datepicker
- CSS (Градиенты, Flexbox)
